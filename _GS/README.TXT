설치방법:
--------------------------------------------------------------------------
GS9706.EXE는 자동 풀림 압축파일입니다.적당한 디렉토리(폴더)에서 압축을 
푼후 생성된 파일 중 INSTALL.BAT를 실행하면 모든 설치가 끝납니다.

                       전체적인 기능 소개
--------------------------------------------------------------------------
1. 메뉴 인터페이스
  WINDOWS 95에서 사용되는 마우스 드래깅 방식과 호환되는 편리한
  메뉴 및 선택 박스, 도움말 박스 사용.

2. 일반 통신 측면
   (1) 모뎀 속도: 1200BPS - 115200BPS
   (2) 모뎀 포트: COM1 COM2 COM3 COM4
   (3) 자동 전화 걸기 기능
   (4) 단축 매크로 기능
   (5) 문서 갈무리 기능(GSM.EXE)
   (6) 지나간 화면 보기 기능
   (7) 지나간 화면 저장 기능(GSM.EXE)
   (8) 채팅 윈도우 기능(GSM.EXE)
   (9) DOSKEY 형식의 입력창(GSH.EXE)
  (10) 마우스 클릭을 통한 화면상의 문자열 송신 기능
  (11) 두벌식, 세벌식 한글 자판 지원
  (12) 통신 화면 색상 선택 기능
  (13) 프로토콜 지원.(텍사스 Z모뎀)

3. 바둑 통신 측면
   (1) 두 가지 통신 프로그램 제공
       1) GSM.EXE(범용 통신 바둑)
          하이텔(신규 & 구 대국실 모두 지원), 천리안, 나우누리 대국실 동시 지원
       2) GSH.EXE(국지용 통신 바둑)
          하이텔 구 대국실만 지원
   (2) 수읽기 기능
       통신 대국 도중 깊은 수읽기가 필요할 경우 직접 화면 위에
       착점을 하며 수읽기를 할 수 있는 기능
   (3) 자체 계가 기능
       호스트(나우누리, 하이텔, 천리안, NICE바둑)의 통합 계가와는 별도로
       기성 프로그램 내부에서 지원하는 자체 계가 기능으로 바둑이 마무리되
       지 않은 상태에서도 자체 계가 기능 내부에서 지원하는 착점과 사석 지
       움, 빅 표시를 거쳐 100% 정확한 계가를 할 수 있음.
   (4) 바둑판 고르기 기능
       나뭇결 바둑판을 포함한 3가지 바둑판 제공
   (5) 바둑 화면 전환
       그래픽 바둑 화면과 일반 통신 바둑 화면의 자유로운 전환
   (6) 마우스 착점
       마우스를 이용한 착점 및 사석과 빅좌표 지정
   (7) 명령 송신
       대국실 명령어와 화면에  표시된 문자를 간단한 마우스 조작으로 호스
       트측으로 송신하는 기능
   (8) 통신 게시판을 이용한 바둑 강좌 및 기보 감상 기능(GSM.EXE)
       강력한 게시판 지원 기능으로 통신 게시판의
       게시물을 검색하는 행위만으로 그 즉시 기보나 바둑 강좌의 감상이
       가능함. 마찬가지로 게시판 기보나 바둑 강좌물도 기보 갈무리 기능으로
       역으로 갈무리가 가능함.
   (9) 바둑 문서 보기 기능(GSM.EXE)
       이야기6.1 이상 바둑상태 화면과 호환되는 바둑판 바둑알(일반착점, 영
       문착점(a-z), 숫자착점(1-15))을 포함한 텍스트 바둑 문서의 통신상의
       보기 기능
  (10) 대국실 메시지 창 좌우 스크롤 기능
       범용 대국실 지원 통신 프로그램 GSM.EXE가 진보된 구조로 설계된 통신
       바둑 프로그램임 단적으로 증명하는 기능.  나우누리, 하이텔, 천리안,
       NICE바둑의 각 대국실에서 우측에 수신되는 메시지의 위치가 저마다
       서로 틀릴지라도 사용자가 임의로 수신 범위를 조정할 수 있는 특징이
       있음. 화면 하단의 ◀▶ 표시를 마우스로 클릭하면서 실제로 확인해
       보기 바람.
  (11) 마우스 착점
       마우스를 이용한 착점 지원 대형 바둑판은 물론 소형 바둑판에서도
       마우스 착점 지원
  (12) 키보드의 화살표 키를 이용한 착점 기능 (GSM.EXE)
       [97.02]버젼까지 착점 키는  CTRL + ENTER 였는데 이 부분에 대한
       불편함이 있어서 [97.06]부터는 단순히 ENTER 키만으로도 착점이 가능
       하게 했습니다.
  (13) 명령 송신
       대국실 명령어와 화면상에 표시된 문자를 간단한 마우스 조작으로
       호스트 측으로 송신하는 기능
  (14) 기보 갈무리 기능
       편하고 강력한 기보 갈무리 기능. 이야기 프로그램의 일반 갈무리 형식
       과 동일한 방법으로 기보 갈무리를 지원. 대형 BBS 기보실에 등록돼 있
       는 수 천 종류의 프로 기사 기보와 일반 대국자 기보를 편리하게 갈무리
       할 수 있음. (하나의 기보 파일에 여러 개의 기보를 동시에 저장할
       수도 있음. 하이텔의 경우 해설이 첨부된 프로 기사의 기보는 해설까지
       함께 갈무리가 가능하고 갈무리된 기보는 기성의 기보 관리 프로그램
       GST.EXE에서 감상할 수 있음)
       게시판을 지원용 기보나 바둑 강좌도 동일한 요령으로 갈무리가 가능함.

4. 기보 관리 측면
   (1) 기보 감상 기능
      기성의 바둑 통신 기능으로 갈무리된 기보와 기성의 기보 입력 도구로
      작성된 기보를 다시 보여 주는 기능.  편리한 기보 감상을 위한 여러
      가지 도구를 탑재하고 있음.
      1) 수동 및 자동 감상(1-5초) 기능
      2) 한수 후진 기능
      3) 임의의 수 후진 기능
      4) 임의의 수 전진 기능
      5) 기보간의 페이지 이동 기능
      6) 자체 계가 기능
      7) 수 읽기 기능
      8) 진행 번호 표시 기능 등등...
   (2) 기보 입력 기능
       생물처럼 살아 있는 듯한 기보 작성을 위한 최적의 도구를 제공.
       기보 입력 도구의 소개는 생략. 기보 입력 도움말을 참고.
       (데모용으로 작성된 9706_?.GSB 기보 파일을 감상해 보기 바람)
   (3) 기보 호출
       미쳐 다 입력하지 못한 기보를 호출해서 나머지 부분을 입력할 수 있는
       기능.
   (4) 기보 편집 기능
       기존의 기보에 해설을 포함한 변화도를 삽입 또는 삭제할 수 있는 기능
   (5) 정석 사전 기능
       다중 트리 형태로 가공된 총 5474 종류의 방대한 정석 데이타를 이용한
       정석 사전 기능
   (6) 바둑 문서 생성 기능
       현재 기성 화면에 표시된 바둑판 상황에 대한 정보(정상착점, 영문착점
       숫자착점, 한글착점 포함)를 텍스트 문서로 저장하는 기능. 저장된 문서
       는 일반 BBS(하이텔, 나우누리, 천리안) 게시판의 게시물로 사용될 수
       있으며... 기성 범용 통신 프로그램 GSM.EXE(96.12)나  이야기6.1 이상
       바둑 화면 상태에서 검색이 가능함 단 이야기 사용자는 기성에서 지원
       하는 한글착점은 볼 수 없음.
   (7) 게시판용 기보 생성 기능
       기성의 기보를 게시판에서 사용할 수 있는 통신 기보로 자동적으로
       변환해 주는 기능.

관전 시 그리고 "기성ANSI"로 작성된 게시판 기보 감상 시에도 해당 기능을
사용할 수 있습니다.

사용했는데 이 부분에 대한 불편을 호소하는 의견이 있어서

--------------------------------------------------------------------------
                         추가된 기능 [97.01]
--------------------------------------------------------------------------
기성[97.01]은 통신 게시판을 이용한 바둑 강좌 및 기보 감상을 지원합니다.
이는 통신 접속 상에서 해당 게시판을 검색하는 행위만으로도 그 즉시 기보
및 바둑 강좌의 감상이 가능하다는 의미입니다.
이것은 기성[96.12]에서 보여주었던(이야기6.1이상 호환) 텍스트 바둑 문서의
정적인 검색과는 차원이 틀린 기능으로 바둑만을 위해 정의된 기성ANSI를
이용한 동적인 바둑 화면 구성입니다.

물론 일반 텍스트 바둑 문서의 정적인 검색도 계속 지원합니다.

게시판에 사용되는 게시물의 형식:
게시판 기보 감상의 효과적인 지원을 위해 게시판에 등록되는 게시물은
기본적으로는 기성[96.12]이상에서 사용되는 기보 파일을 사용합니다.
따라서 일반 착점은 물론이고 영문/한글/숫자/기호/보관/복구/청소/공배착점
으로 명칭된 기성의 모든 특수 착점과 해설을 그대로 사용할 수 있습니다.
그리고 기보 편집 기능으로 삽입된 변화도 블럭 역시 모두 수용을 합니다.
그외 기보 입력 시 착점이 아닌 옵션으로 주어진 번호표시 기능은
[97.02]버젼까진 통신상에서는 아무런 역할도 하지 않았으나
이번 [97.06]부터는 역시 해당 기능도 모두 수용합니다.

기성의 기보를 통신 게시판에 올리기 위해서는 통신 환경만의 특수성을
감안한 별도의 처리를 거쳐야 하며... 그 모든 처리는 기보관리(GST.EXE)
프로그램 내부의 [게시판 기보] 메뉴에서 지원합니다.

게시판용 기보 역시 감상 도중 다시 역으로 갈무리 할 수 있습니다.

[info]
본 기능을 잘 활용하면 상당히 효과적인 통신 바둑 문화의 공유가 가능할
거라고 봅니다. 게시판의 가장 큰 장점은 별도의 허락 과정 없이 사용자가
바로 쓰기를 할 수 있다는 점입니다.
게시판을 통한 기보와 바둑 강좌의 지원 기능을 활용한다면  대형 호스트
에서만 지원이 가능한 바둑의 온라인 생중계와 비슷한 혹은 어떤 면에서
더 다양한 효과를 거둘 수 있다고 봅니다.

그 외 [96.12]에서 발견된 4 가지 오류를 수정했습니다.
1. 국지용(GSH.EXE)통신의 수읽기 착점의 순번 문제.
2. 국지용(GSH.EXE)통신의 자체 계가에서 사석 지정 문제
3. 범용(GSM.EXE)통신의 기보 갈무리 마지막 수 처리 문제
4. 기보관리(GST.EXE)의 기보 편집 기능 사용 중 페이지 이동 문제.

--------------------------------------------------------------------------
                  시스템 구비 조건 & 설치 방법
--------------------------------------------------------------------------
VIDEO CARD  : VGA 이상
CPU         : 386-SX 이상
RAM         : 최소 1MByte 이상, 권장 2MByte 이상
MOUSE       : 필수
MODEM       : 통신 기능을 사용할 때 필수
OS          : DOS 3.1이상, Windows95
              Windows95에서 기성 사용시 화면 깜빡임이 나타나며 한글
              입력 장애가 나타나는 시스템을 보유하신 분은 기성의
              효과음 설정을 끄고 사용하시기 바랍니다.

GS9706.EXE는 자동 풀림 압축 파일입니다.
기성 프로그램을 설치하고자 하는  임의의 디렉토리를 만들고 해당 디렉토리에
GS9706.EXE를 복사한 후 GS9706.EXE <ENTER>  명령을 내리면 압축이 풀리고
설치가 끝납니다.
설치 후 GS.EXE를 실행시킨 후 환경 설정 메뉴에서 통신 포트, 모뎀 속도, 초기
화 명령, 한글 자판, 바둑판 종류, 효과음 사용 여부, 통신 터미널 색상 등을
원하는 값으로 설정해 주기 바랍니다.
